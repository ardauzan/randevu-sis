# SDU Randevu Sistemi

## Kurulum

Bu proje bun ile çalışır ve versiyon 1.1.6 ile test edilmiştir

```bash
curl -fsSl https://bun.sh/install | bash -s "bun-v1.1.6"
```

Sonrada projenin dizinine girip aşağıdaki komutu çalıştırın

```bash
bun install
```
